# Authorization
- CISSP Glossary
<br> = The process of defining the specific resources a user needs and determining the type of access to those resources the user may have.

  <br>=กระบวนการที่ใช้กำหนดทรัพยากรเฉพาะที่ผู้ใช้ต้องการ และพิจารณาประเภทของการเข้าถึงทรัพยากรเหล่านั้นที่ผู้ใช้อาจมีสิทธิ์ได้รับ
- Gemini
  <br>=การกำหนดทรัพยากรเฉพาะเจาะจงที่ผู้ใช้จำเป็นต้องใช้ และกำหนดประเภทของการเข้าถึงทรัพยากรเหล่านั้นที่ผู้ใช้สามารถมีได้

- Gartner
  <br>=A process ensuring that correctly authenticated users can access only those resources for which the owner has given them approval.
- ChatGPT

- Gemini
  <br>=กระบวนการที่มั่นใจได้ว่าผู้ใช้ที่ได้รับการตรวจสอบยืนยันตัวตนอย่างถูกต้องสามารถเข้าถึงเฉพาะทรัพยากรที่เจ้าของได้อนุญาตให้พวกเขาเท่านั้น

- ความหมายที่สรุปเอง
  <br>=เป็นกระบวนการที่รับรองว่าผู้ใช้ได้รับการยืนยันตัวตนและสามารถเข้าถึงทรัพยากรที่มีสิทธื์ได้รับอย่างถูกต้อง

- ตัวอย่างในชีวิตประจำวัน
  <br>=การเข้าใช้ระบบ Wi-Fi โรงแรมหรือร้านกาแฟอาจให้รหัส Wi-Fi ที่มีสิทธิ์จำกัด เช่น ใช้เฉพาะการเข้าถึงอินเทอร์เน็ต แต่ไม่สามารถเข้าถึงการตั้งค่าภายในเครือข่ายของร้านได้
